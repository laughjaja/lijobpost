# LinkedIn CNC Job Carousel

Reusable HTML/CSS carousel for CNC roles.

## What is inside

- `index.html` - editable slide content
- `styles.css` - typography, spacing, colors, and layout

## How to edit

Open `index.html` and change the job titles, locations, pay, and fit lines.

Open `styles.css` to adjust fonts, sizing, colors, and layout.

## Export workflow

1. Open `index.html` in a browser.
2. Use browser print or screenshot/export each slide at 1080 x 1080.
3. Upload the exported images to LinkedIn as a carousel.
